# stock-market-api
Developed a RESTFUL backend API using FastAPI to manage stock market data with secure user authentication. The system supports CRUD operations,JWT-based login,
and database integration using SQLite.
# Problem statement
Retail investors and learners need a secure and structured backend system to manage stock data and user authentication efficiently.
# Solution
Build a modular backend system that:
Implements JWT authentication
Stores stock data securely
Provides REST APIs
Users Swagger UI for testing
